Bienvenue sur le dépôt de mon portfolio interactif en 3D, présentant mes travaux et projets de recherche en modélisation, calcul haute performance (HPC), et simulation (CFD).

🔗 **[Visiter le Portfolio en Ligne](https://Chaz024.github.io/Portfolio_CHAZALON/)**

## À propos de ce projet

Ce site a été conçu pour offrir une expérience "Premium" et fluide, mettant en valeur des projets d'ingénierie complexes :
- **Rendu 3D temps réel** : Modèle interactif d'A380 avec reflets métalliques réalistes, matériaux PBR et lignes de courant intégrées.
- **Performances et Fluidité** : Génération statique ultrarapide, sans aucun cadriciel lourd.
- **Transitions cinématographiques** : Pool aléatoire d'animations "Glassmorphism" et "Sweep" pour une navigation professionnelle.

## Stack Technique

- **Astro** (Génération statique et View Transitions API)
- **React Three Fiber & Drei** (Rendu 3D WebGL via Three.js)
- **Framer Motion** (Animations et micro-interactions)
- **CSS Vanilla** (Variables CSS, Typographie système, pas de Tailwind)

## Lancement en local

Si vous souhaitez explorer le code en local :

```sh
npm install
npm run dev
```
Le site sera accessible sur `http://localhost:4321`.
